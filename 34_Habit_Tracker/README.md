A Python module that interacts with the [Pixela API](https://pixe.la/) to track daily habits and visualize progress. This script allows you to create a personalized tracking graph and log your daily data using standard HTTP requests.

This project demonstrates the use of advanced REST API methods (`POST`, `PUT`, `DELETE`), dynamic endpoint routing, date formatting, and secure environment variable management.

## ✨ Features

* **User Management:** Create a new Pixela user account directly via API.
* **Graph Generation:** Setup and personalize custom tracking graphs (e.g., hours coded, pages read).
* **Data Entry (POST):** Add new daily pixels dynamically using `datetime` to log your habits.
* **Data Modification (PUT):** Update existing pixel values for any given date in the past.
* **Data Deletion (DELETE):** Remove specific pixels to correct mistakes or clear data.
* **Secure Configurations:** Uses `python-dotenv` to keep the API token and username safely hidden, utilizing custom variables to prevent OS environment clashes.

---

## 🛠️ Technologies & Libraries

* **Python 3.x**
* `requests` (for handling HTTP `GET`, `POST`, `PUT`, and `DELETE` requests)
* `datetime` (for dynamic date formatting and targeting specific records)
* `os` & `python-dotenv` (for secure environment variable extraction)
