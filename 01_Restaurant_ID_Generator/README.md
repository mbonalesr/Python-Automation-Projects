# 🆔 Restaurant ID Generator

### Project Description
A Python script designed to automate the generation of unique identifiers for restaurant employees or workstations. This project focuses on string manipulation and basic input handling to create standardized formatted IDs.

### Key Concepts
* **Input Handling:** Capturing user data via console.
* **String Manipulation:** Concatenation and methods (e.g., capitalization).
* **Variables:** Storing and retrieving data for dynamic formatting.

### Usage
Run the script and follow the prompts to enter the employee's name and section to generate their unique ID badge code.
