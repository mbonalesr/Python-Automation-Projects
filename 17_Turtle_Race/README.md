# 🐢 Turtle Race Betting Game

A fun, interactive Python script that simulates a turtle race using the standard `turtle` library. Place your bet on a colored turtle and watch them race across the screen using randomized speeds!

## 🚀 Features
* **Interactive UI:** Prompts the user to place a bet before the race begins.
* **Object-Oriented Programming (OOP):** Instantiates multiple `Turtle` objects with unique states (colors and starting positions).
* **Randomized Movement:** Uses Python's `random` module to ensure every race is unique and unpredictable.
* **Win/Loss Detection:** Automatically detects when a turtle crosses the finish line based on Cartesian coordinates and evaluates the user's bet.

## 🛠️ Technologies Used
* **Python 3.x**
* **Turtle Graphics** (Python standard UI library)
* **Random Module** (For randomized distance generation)
