## 📌 Overview
A lightweight, user-friendly desktop application built with Python and Tkinter that instantly converts distances from Miles to Kilometers. This project demonstrates the implementation of Graphical User Interfaces (GUI), widget management, and dynamic data updating using Python.

## ✨ Features
* **Clean Interface:** Uses a structured grid layout for a neat and intuitive user experience.
* **Real-Time Calculation:** Converts input values on the fly with a single button click.
* **Precision:** Automatically rounds the output to one decimal place for easy readability.
* **Custom Styling:** Implements the modern `Lexend` font for a sleek look.

## 🛠️ Technologies Used
* **Python 3.x:** Core programming language.
* **Tkinter:** Standard GUI library for Python used to create the window, labels, entry fields, and buttons.
