## About The Project
Quizzler is a desktop trivia application built with Python. It features a graphical user interface (GUI) created with Tkinter and tests your general knowledge with a variety of True/False questions.

## Features
* **Interactive UI:** Gives immediate visual feedback (flashes green for correct answers, red for incorrect).
* **Score Tracking:** Keeps a running tally of your score throughout the session.
* **API Integration:** Connects to the OpenTDB API to retrieve real-time trivia data and generate the question bank.
* **Offline Fallback:** Includes built-in error handling to load a local set of emergency questions if the internet connection drops, preventing application crashes.
