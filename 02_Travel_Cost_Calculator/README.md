# ⛽ Travel Cost & Bill Splitter

### Project Description
An automated calculator to estimate logistical costs for group travel. The program takes inputs such as fuel price, distance, consumption, and number of passengers to determine the exact cost per person.

### Key Concepts
* **Data Type Conversion:** Handling `strings` to `floats` for mathematical operations.
* **Arithmetic Logic:** Implementing formulas for fuel efficiency and cost division.
* **F-Strings:** Dynamic output formatting for currency and results.

### Usage
Input the total distance, current gas price, and number of travelers to receive a breakdown of the total trip cost and individual share.
