# ✂️ Rock, Paper, Scissors (Logic Engine)

### Project Description
A classic game implementation playing against the Computer. This project demonstrates control flow logic and randomization handling. It includes validation to prevent invalid user inputs and determines the winner based on game rules.

### Key Concepts
* **Randomization:** Using the `random` module to generate computer choices.
* **Control Flow:** Complex nested `if/elif/else` statements to determine game outcomes.
* **List Indexing:** Mapping user numeric input to game assets (Rock, Paper, or Scissors).
* **Error Handling:** Managing out-of-range inputs.

### Usage
Run the game, type `0` for Rock, `1` for Paper, or `2` for Scissors, and see if you can beat the computer logic.
