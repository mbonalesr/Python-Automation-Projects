# 🎨 Hirst Painting Generator

A Python script that recreates Damien Hirst's famous spot paintings using the `turtle` graphics library. The script generates a perfect 10x10 grid of colored dots, randomly selecting colors from a pre-defined palette extracted from a real Hirst painting.

## 🚀 Features
* Generates a 10x10 grid of perfectly spaced dots.
* Uses a custom RGB color palette extracted from a reference image.
* Implements nested loops for efficient 2D drawing coordinates.
* Configured with optimal drawing speed and hidden cursor for a clean output.

## 🛠️ Technologies Used
* **Python 3.x**
* **Turtle Graphics** (Python standard library)
* **Colorgram.py** *(Used during development to extract the RGB values from the original image)*
